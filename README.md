# Vylux-ResourcePack

Плагин для автоматической выдачи resource pack игрокам с поддержкой версий от **1.19.4 до 1.21.11** (Paper/Purpur).

Разработан для проекта **Vylux**.

---

## ⚡ Возможности

* Автоматическая отправка resource pack при входе и смене мира
* Поддержка нескольких миров
* Полная поддержка новых API (UUID + hash)
* Корректная работа с версиями 1.21.11+
* Поддержка `force` режима
* Удаление только **своего** resource pack
* Fallback система (команда / телепорт / кик)
* Кэширование состояния игроков
* Безопасная работа с reload
* Минимальный debug режим

---

## 📦 Поддержка версий

* ✔ 1.19.4
* ✔ 1.20.x
* ✔ 1.21.x
* ✔ 1.21.11+

---

## ⚙️ Установка

1. Скопируйте `.jar` файл в папку `/plugins`
2. Перезапустите сервер
3. Отредактируйте `config.yml`
4. Выполните:

```
/vrp reload
```

---

## ⚙️ Конфигурация

```yaml
resource-pack-url: "https://example.com/pack.zip"
resource-pack-uuid: "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"
resource-pack-hash: "SHA1_HASH_40_SYMBOLS"

force: true

world:
  - world
  - spawn

command-noresource: "spawn %player%"

remove-pack-url: ""

debug: false
```

---

## 🔐 Важно (1.21.11+)

Для корректной работы обязательно:

* Указывать **SHA-1 hash**
* Использовать **валидный HTTPS URL**
* Следить, чтобы файл не менялся (или обновлять hash)

---

## 🧠 Поведение

### Если `force: true`

* Игрок обязан принять пак
* При отказе — клиент сам разрывает соединение

### Если `force: false`

* При отказе выполняется:

  * команда
  * или fallback (телепорт / кик)
